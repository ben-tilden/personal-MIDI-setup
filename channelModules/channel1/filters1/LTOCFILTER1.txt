# launchkey to channel filter

# filter out messages that don't have CC or FC as their routing (FF)

# messages expected:

# BLOCK | F0 FF FF 00 00 F7 - message sent from launchkey to xtouch to reset lighting
# ALLOW | F0 CC FF 00 00 F7 - message sent from launchkey to channel to reset channel
# BLOCK | F0 FF FF CC CC F7 - message sent from launchkey to xtouch at change channel - send xtouch lights to xtouch for new channel

# ALLOW | F0 FC    00    val  F7      currentChannel           
# ALLOW | F0 CC    01    val  F7      numPressed                  
# BLOCK | F0 FF/11 0A    val  F7      setPhrase                
# ALLOW | F0 CC    0B    val  F7      phraseVal                
# BLOCK | F0 FF    0C    val  F7      phraseIndex              
# ALLOW | F0 CC    0D    val  F7      phraseLengthPlusOffset   
# ALLOW | F0 CC/FF 0E    val  F7      gateOpened               
# ALLOW | F0 CC    index val1 val2 F7 phraseVal                
# ALLOW | F0 CC 20 val1 val2 val3 val4 val5 val6 val7 val8 val9 val10 F7 arpValues

# BLOCK | any channel specific note and CC messages

  IF M1 > FC
    BLOCK
    EXIT
  END
