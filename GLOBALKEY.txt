# global array key

# global array will be set in the following way
# G0 - channel
# G1 - arpPhraseToggle
# G2 - arpPhraseOnOffToggle
# G3 - tempoMode
# G4 - tempoModeMod
# G5 - tempoPulse
# G6 - gate
# G7 - arpValueMode
# G8 - arpValueModeMod
# G9 - setPhrase
# GA - waveguideToggle
# GB - drumParams (Part 1)
# GC - drumParams (Part 2)
# GD - drumParams (Part 1+2)
# (GX)(whatever is the last param) - arp values
# (GY)(whatever is the last param) - arp order pressed
# (GZ)(whatever is the last param) - phrase sequence
