# global array will be set in the following way
# G0 - channel
# G1 - numPressed
# G2 - arpPhraseToggle
# G3 - arpPhraseOnOffToggle
# G4 - tempoModeMod
# G5 - tempoPulse
# G6 - gate
# G7 - arpValueModeMod
# G8 - setPhrase
# G9 - waveguideToggle
# GA - drumParams (Part 1)
# GB - drumParams (Part 2)
# GC - drumParams (Part 1+2)
# GD - velocity
# GE - gateOpened
# GF - phraseVal
# G10 - phraseLengthPlusOffset
# G20 - G29 - arp values        (G10 G11 G12 G13 G14 G15 G16 G17 G18 G19)
# G2A - G39 - phrase sequence      (G24 G25 G26 G27 G28 G29 G2A G2B G2C G2D G2E G2F G30 G31 G32 G33 ETC.)
